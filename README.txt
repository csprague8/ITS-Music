Setup:
    Before first use, make sure the visual studio 2013 redistributable 
    package is installed. To do this run vcredist_x64.exe on your machine.

Features:
    * Note Lengths
        * Whole  notes
        * Half notes
        * Quarter notes
        * Eighth notes
        * Sixteenth notes
    * Note Values
    * Note Clefs
        * Treble Clef
        * Bass Clef
    * Rest Types
        * Whole notes
        * Half notes
        * Quarter notes
        * Eighth notes
        * Sixteenth rests
    * Rhythms
        * 13 Pre-made Rhythm Patterns
    * Scales
        * 49 Scales including Major, minor, and modes
    * Music keys
    * Note mapping
        * mapping notes to the fretboard of a guitar
    * Adjustable Metronome

Operation:
    * Start the application
    * Select a category
    * If available, select a subcategory
    * If available, select a base note for the displayed information
